## Программное обеспечение

- **Go 1.21+** (для сборки из исходного кода)
- **Токен Telegram-бота** (создайте через [@BotFather](https://t.me/BotFather))

Других зависимостей нет. BotMux — это единый бинарный файл, не требующий внешних сервисов.

## Сеть

| Порт | Протокол | Назначение |
|------|----------|------------|
| 8080 | HTTP | Веб-интерфейс и REST API |
| — | HTTPS (исходящий) | `api.telegram.org` (опрос для каждого бота) |
| — | HTTP/HTTPS (исходящий) | URL бэкенда (если включён режим прокси) |
| — | HTTP/HTTPS (исходящий) | Эндпоинт LLM API (если включена LLM-маршрутизация) |
