BotMux собирает все сообщения и публикации каналов, видимые боту, в реальном времени.

## Возможности

- **Сбор в реальном времени** всех сообщений, доступных боту
- **Полнотекстовый поиск** по истории сообщений
- **Постраничная лента** с отправителем, временем и содержанием
- **Ответ на сообщения** — нажмите кнопку ответа для отправки ответа в контексте
- **Визуализация цепочек ответов** — сообщения показывают плашку "Ответ на @user" с прокруткой по клику
- **Автоматическое обнаружение чатов** — бот определяет группы, в которые его добавили
- **Отслеживание удалённых сообщений** — удалённые сообщения визуально отмечаются в интерфейсе

## Поддержка медиа

Сообщения с медиа отображаются встроенно:

| Тип | Отображение |
|-----|-------------|
| Фотографии | Встроенное изображение с лайтбоксом (клик для увеличения) |
| Видео | Встроенный видеоплеер |
| Анимации (GIF) | Встроенная анимация |
| Стикеры | Конвертация WebP→PNG для совместимости с браузерами |
| Голосовые/Аудио | Встроенный аудиоплеер |
| Документы | Ссылка для скачивания |
| Видеозаметки | Встроенный плеер |
