---
title: "Журнал аудита"
description: "Отслеживание всех действий администратора через веб-интерфейс"
---

Каждое действие администратора, выполненное через BotMux, записывается в журнал аудита.

## Отслеживаемые действия

- Блокировки и разблокировки
- Удаления и закрепления сообщений
- Повышения и понижения администраторов
- Изменения титулов
- Назначения тегов

## Поля записи

| Поле | Описание |
|------|----------|
| Действие | Тип действия администратора |
| Исполнитель | Пользователь, выполнивший действие |
| Объект | Затронутый пользователь или сообщение |
| Детали | Дополнительный контекст |
| Время | Когда было выполнено действие |

## API

```
GET /api/adminlog?chat_id=&limit=&offset=
```

Возвращает постраничные записи журнала аудита для указанного чата.
