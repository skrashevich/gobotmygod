---
title: "Теги пользователей"
description: "Система пользовательской классификации участников чатов"
---

Теги предоставляют гибкий способ классификации и маркировки участников чатов.

## Возможности

- Назначение произвольных тегов пользователям (VIP, Модератор, Спамер, Доверенный и т.д.)
- **Цветовая маркировка** для визуального различия
- **Несколько тегов** на пользователя
- **Для каждого чата** — один и тот же пользователь может иметь разные теги в разных чатах

## API

| Метод | Эндпоинт | Описание |
|-------|----------|----------|
| GET | `/api/tags?chat_id=` | Получить все теги чата |
| GET | `/api/tags/user?chat_id=&user_id=` | Получить теги пользователя |
| POST | `/api/tags/add` | Добавить тег (JSON-тело) |
| POST | `/api/tags/remove?id=` | Удалить тег по ID |

### Тело запроса добавления тега

```json
{
  "bot_id": 1,
  "chat_id": -1001234567890,
  "user_id": 123456789,
  "username": "johndoe",
  "tag": "VIP",
  "color": "#4CAF50"
}
```
