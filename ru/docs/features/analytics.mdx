BotMux предоставляет аналитику для каждого чата, доступную из детального просмотра чата.

## Метрики

- **Общее количество сообщений** — все сообщения за время существования чата
- **Дневная активность** — сообщения за сегодня
- **Активные пользователи за неделю** — уникальные отправители за последние 7 дней

## Графики

- **Почасовое распределение активности** — суточный график частоты сообщений за 7-дневное окно

## Рейтинг

- **Топ-10 участников** — самые активные пользователи по количеству сообщений
