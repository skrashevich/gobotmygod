BotMux позволяет выполнять действия администратора Telegram из веб-панели. Все действия требуют наличия соответствующих прав администратора у бота в чате.

## Доступные действия

### Сообщения
- **Отправка сообщений** — рассылка в любой чат с HTML-форматированием (`<b>`, `<i>`, `<code>`, `<a>`)
- **Закрепление / открепление сообщений**
- **Удаление сообщений**

### Пользователи
- **Блокировка / разблокировка пользователей**

### Администраторы
- **Просмотр всех администраторов** с разбивкой по разрешениям
- **Повышение пользователей** до администратора с гранулярными разрешениями:
  - Управление чатом, Удаление сообщений, Ограничение участников
  - Приглашение пользователей, Закрепление сообщений, Изменение информации о чате
  - Повышение других участников
- **Изменение разрешений** существующих администраторов
- **Установка пользовательских титулов** (до 16 символов)
- **Понижение** администраторов

## Журнал аудита

Каждое действие администратора, выполненное через веб-интерфейс, записывается:
- Блокировки и разблокировки
- Удаления и закрепления сообщений
- Повышения и понижения администраторов
- Изменения титулов
- Назначения тегов

В журнале отображаются тип действия, исполнитель, объект, детали и время.
