BotMux отображает медиаконтент встроенно в ленте сообщений.

## Поддерживаемые типы

| Тип | Отображение | Примечания |
|-----|-------------|------------|
| Фотографии | Встроенное изображение с лайтбоксом | Клик для просмотра в полном размере |
| Видео | Встроенный видеоплеер | |
| Анимации | Встроенный GIF | |
| Стикеры | Встроенное изображение | Автоконвертация WebP→PNG |
| Голосовые сообщения | Аудиоплеер | |
| Аудио | Аудиоплеер | |
| Документы | Ссылка для скачивания | |
| Видеозаметки | Видеоплеер | Круглые видеосообщения |

## Конвертация стикеров

Стикеры Telegram используют формат WebP, который может не поддерживаться во всех браузерах. BotMux автоматически конвертирует стикеры из WebP в PNG с помощью чистого Go-кодека WebP.

Для анимированных стикеров (формат TGS) BotMux использует миниатюру стикера (статический превью) вместо основного файла.

## Лайтбокс изображений

Нажмите на любую фотографию, чтобы просмотреть её в полном размере в оверлее. Закрытие по клику или клавише Escape.

## Прокси медиа

Медиафайлы обслуживаются через `/api/media?file_id=&bot_id=`, который проксирует загрузки из Telegram с автоматической конвертацией формата.
