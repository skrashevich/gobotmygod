---
title: "Маршрутизация между ботами"
description: "Маршрутизация сообщений между ботами на основе настраиваемых условий"
---

BotMux может маршрутизировать сообщения между ботами на основе настраиваемых правил.

## Типы условий

| Тип | Описание |
|-----|----------|
| **Совпадение текста** | Regex-шаблон для текста сообщения (без учёта регистра) |
| **ID пользователя** | Сообщения от определённого пользователя Telegram |
| **ID чата** | Все сообщения из определённого чата/группы |
| **LLM** | Маршрутизация на основе ИИ — опишите правило на естественном языке |

## Действия

| Действие | Описание |
|----------|----------|
| **Forward** | Отправить текст сообщения как новое сообщение через целевого бота |
| **Copy** | Переслать оригинальное сообщение с сохранением авторства через `forwardMessage` |
| **Drop** | Молча проигнорировать сообщение |

## Обратный путь Source-NAT

Ответы на маршрутизированные сообщения автоматически отправляются обратно через исходного бота в исходный чат, сохраняя контекст разговора. Каждый ответ создаёт новое сопоставление, обеспечивая непрерывные межботовые диалоги.

## Защита от циклов

Сообщения от ботов не подвергаются обратной маршрутизации, что предотвращает бесконечные циклы.

## Управление

Правила маршрутизации управляются для каждого бота из веб-интерфейса с возможностью включения/отключения.
