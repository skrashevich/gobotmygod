## Создание бота

1. Откройте [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot` и следуйте инструкциям
3. Скопируйте токен бота

## Настройка режима конфиденциальности

По умолчанию боты видят только сообщения, которые их упоминают или начинаются с `/`. Чтобы видеть все сообщения группы:

1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте `/setprivacy`
3. Выберите вашего бота
4. Выберите **Disable**

<Warning>
Изменения режима конфиденциальности применяются только к группам, в которые бот вступит **после** изменения. Удалите и заново добавьте бота в существующие группы.
</Warning>

## Необходимые разрешения

Для полной функциональности назначьте бота администратором с этими разрешениями:

| Разрешение | Для чего нужно |
|------------|----------------|
| Удаление сообщений | Удаление сообщений из интерфейса |
| Блокировка пользователей | Действия блокировки/разблокировки |
| Закрепление сообщений | Закрепление/открепление сообщений |
| Приглашение пользователей | Управление ссылками-приглашениями |
| Добавление администраторов | Повышение/понижение администраторов |
| Изменение информации о группе | Изменение настроек чата |

Бот работает с любым набором разрешений — функции, требующие отсутствующих разрешений, будут возвращать ошибки.

## Добавление в чат

1. Добавьте бота в вашу Telegram-группу или канал
2. Назначьте его администратором с указанными выше разрешениями
3. Чат автоматически появится в боковой панели BotMux
