## 创建机器人

1. 在 Telegram 中打开 [@BotFather](https://t.me/BotFather)
2. 发送 `/newbot` 并按照提示操作
3. 复制机器人令牌

## 配置隐私模式

默认情况下，机器人只能看到提及它或以 `/` 开头的消息。要查看所有群组消息：

1. 打开 [@BotFather](https://t.me/BotFather)
2. 发送 `/setprivacy`
3. 选择你的机器人
4. 选择 **Disable**

<Warning>
隐私模式的更改仅适用于更改**之后**加入的群组。请将机器人从现有群组中移除并重新添加。
</Warning>

## 所需权限

为获得完整功能，请将机器人设为管理员并授予以下权限：

| 权限 | 用途 |
|------------|-------------|
| 删除消息 | 从 UI 中删除消息 |
| 封禁用户 | 封禁/解封操作 |
| 置顶消息 | 置顶/取消置顶消息 |
| 邀请用户 | 邀请链接管理 |
| 添加新管理员 | 提升/降级管理员 |
| 修改群组信息 | 修改聊天设置 |

机器人可以在部分权限下工作——缺少权限的功能将返回错误。

## 添加到聊天

1. 将机器人添加到 Telegram 群组或频道
2. 将其提升为管理员并授予上述权限
3. 聊天会自动出现在 BotMux 侧边栏中
