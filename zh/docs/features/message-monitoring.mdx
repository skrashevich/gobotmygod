BotMux 实时采集机器人可见的所有消息和频道帖子。

## 功能

- **实时采集**机器人可见的所有消息
- **全文搜索**消息历史
- **分页消息流**，包含发送者、时间戳和内容
- **回复消息** — 点击回复按钮可在上下文中发送回复
- **回复链可视化** — 消息显示"回复 @user"标签，点击可滚动定位
- **自动发现聊天** — 机器人会检测它被添加到的群组
- **软删除追踪** — 已删除的消息在 UI 中会有视觉标记

## 媒体支持

带有媒体的消息会内联渲染：

| 类型 | 渲染方式 |
|------|-----------|
| 图片 | 内联图片，支持灯箱效果（点击放大） |
| 视频 | 嵌入式视频播放器 |
| 动画 (GIF) | 内联动画 |
| 贴纸 | WebP→PNG 转换，确保浏览器兼容性 |
| 语音/音频 | 嵌入式音频播放器 |
| 文档 | 下载链接 |
| 视频笔记 | 嵌入式播放器 |
