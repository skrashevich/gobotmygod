---
title: "审计日志"
description: "追踪通过 Web UI 执行的所有管理操作"
---

通过 BotMux 执行的每个管理操作都会记录在审计日志中。

## 追踪的操作

- 封禁和解封
- 消息删除和置顶
- 管理员提升和降级
- 头衔变更
- 标签分配

## 日志条目字段

| 字段 | 说明 |
|-------|-------------|
| 操作 | 管理操作类型 |
| 执行者 | 执行操作的用户 |
| 目标 | 受影响的用户或消息 |
| 详情 | 附加上下文信息 |
| 时间戳 | 操作发生的时间 |

## API

```
GET /api/adminlog?chat_id=&limit=&offset=
```

返回指定聊天的分页审计日志条目。
