---
title: "管理操作"
description: "通过 Web UI 进行聊天管理"
---

BotMux 允许从 Web 仪表盘执行 Telegram 管理操作。所有操作都要求机器人在聊天中拥有相应的管理员权限。

## 可用操作

### 消息
- **发送消息** — 广播到任何聊天，支持 HTML 格式（`<b>`、`<i>`、`<code>`、`<a>`）
- **置顶/取消置顶消息**
- **删除消息**

### 用户
- **封禁/解封用户**

### 管理员
- **查看所有管理员**及其权限详情
- **提升用户**为管理员，支持精细权限控制：
  - 管理聊天、删除消息、限制成员
  - 邀请用户、置顶消息、修改聊天信息
  - 提升其他成员
- **编辑权限** — 修改现有管理员的权限
- **设置自定义头衔**（最多 16 个字符）
- **降级**管理员

## 审计日志

通过 Web UI 执行的每个管理操作都会被记录：
- 封禁和解封
- 消息删除和置顶
- 管理员提升和降级
- 头衔变更
- 标签分配

日志显示操作类型、执行者、目标、详情和时间戳。
