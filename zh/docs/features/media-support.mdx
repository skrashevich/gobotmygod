---
title: "媒体支持"
description: "内联渲染图片、视频、贴纸等媒体内容"
---

BotMux 在消息流中内联渲染媒体内容。

## 支持的类型

| 类型 | 渲染方式 | 备注 |
|------|-----------|-------|
| 图片 | 内联图片，支持灯箱效果 | 点击查看原图 |
| 视频 | 嵌入式视频播放器 | |
| 动画 | 内联 GIF | |
| 贴纸 | 内联图片 | WebP→PNG 自动转换 |
| 语音消息 | 音频播放器 | |
| 音频 | 音频播放器 | |
| 文档 | 下载链接 | |
| 视频笔记 | 视频播放器 | 圆形视频消息 |

## 贴纸转换

Telegram 贴纸使用 WebP 格式，部分浏览器可能不支持。BotMux 使用纯 Go WebP 编解码器自动将贴纸从 WebP 转换为 PNG。

对于动态贴纸（TGS 格式），BotMux 使用贴纸缩略图（静态预览）而非主文件。

## 图片灯箱

点击任何图片即可在覆盖层中查看原图。点击或按 Escape 键关闭。

## 媒体代理

媒体文件通过 `/api/media?file_id=&bot_id=` 提供，该接口代理从 Telegram 下载文件并自动进行格式转换。
