---
title: "机器人间路由"
description: "基于可配置条件在机器人之间路由消息"
---

BotMux 可以基于可配置的规则在机器人之间路由消息。

## 条件类型

| 类型 | 说明 |
|------|-------------|
| **文本匹配** | 对消息文本进行正则表达式模式匹配（不区分大小写） |
| **用户 ID** | 来自特定 Telegram 用户的消息 |
| **聊天 ID** | 来自特定聊天/群组的所有消息 |
| **LLM** | 基于 AI 的路由——用自然语言描述规则 |

## 操作

| 操作 | 说明 |
|--------|-------------|
| **转发** | 通过目标机器人将消息文本作为新消息发送 |
| **复制** | 通过 `forwardMessage` 转发原始消息，保留作者归属 |
| **丢弃** | 静默忽略消息 |

## Source-NAT 回程路径

对已路由消息的回复会自动通过原始源机器人发送回原始聊天，保持对话上下文。每个回复都会创建新的映射，实现持续的跨机器人对话。

## 循环保护

机器人发出的消息不会被反向路由，防止无限路由循环。

## 管理

路由规则通过 Web UI 按机器人管理，支持启用/禁用切换。
